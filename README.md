# Steps to setup locally and run
- git clone https://github.com/shashidhxr/mini-crm.git
- cd mini-crm
- cd ./backend
- npm install
- create .env and define DATABASE_URL
- npm start
- cd ./frontend (in another terminal)
- npm install
- npm run dev
- navigate to http://localhost:5173/
